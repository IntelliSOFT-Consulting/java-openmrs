Appointments: As an OpenMRS Module

This tutorial adds some basic functionality to an existing OpenMRS module version of the appointments manager. To begin this tutorial you need to import the module-appointments project into Eclipse.

1. Adding phone numbers to the data model

  1. Add a new <diff> to sqldiff.xml which adds a column called phone to the appointments table
    • Make sure the diff has a greater version number than the existing one
    • Use an ALTER TABLE statement to add the column
  2. Add a new field called phone to Appointment.java, with a getter and a setter
  3. Add a suitable mapping to Appointment.hbm.xml

Build the module using the ant script and deploy it in OpenMRS to check for problems.

2. Adding view/edit for the new field

  1. Add a text field control to appointmentForm.jsp so phone numbers can be entered
  2. Add column to table in appointmentList.jsp so phone numbers can be viewed

Build the module using the ant script and deploy it in OpenMRS to check for problems.
