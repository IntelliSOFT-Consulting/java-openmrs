Console I/O

This set of exercises covers basic console I/O

1. Hello World

Print a message to the console using System.out.println(...)

Solution: Exercise1.java

2. Star patterns

Using a nested for loop, print the following pattern to the console:

*
**
***
****
*****
******
*******
********
*********

Solution: Exercise2.java

3. Reading user input

Prompt the user for their name, read it from the console, and then display it in a message. The following code shows how to read a line from the console:

InputStreamReader in = new InputStreamReader(System.in);
BufferedReader reader = new BufferedReader(in);
String line = reader.readLine();
Solution: Exercise3.java

4. Power calculator

Create a program which allows a user to input two values (x and y) and then calculates and displays x to the power of y

Solution: Exercise4.java

5. Word length calculator

Create a program which allows a user to input three words, which should be stored in an array. Then calculate and display the total length of all the words together.

Solution: Exercise5.java

6. Input loop

Create a program which continually reads user input from the console, and only stops when the user types "q".

Solution: Exercise6.java
